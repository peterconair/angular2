Create Angular Project with Angular CLI

    1. create angular project by using command :  ng new projectname  
    2. install boot strap in the project : npm install bootstrap@next —save
    3. add  “style” and “script” in angular-cli.json : 

 "styles": [
        "../node_modules/bootstrap/dist/css/bootstrap.css",
        "styles.css"
      ],

      "scripts": [
        "../node_modules/jquery/dist/jquery.js",
        "../node_modules/tether/dist/js/tether.js",
        "../node_modules/bootstrap/dist/js/bootstrap.js"
      ],
